# Contributing

This project welcomes contributions. You may open an issue
to request a new feature or report a bug.

If you plan to make a PR, if it's a small-sized changed,
feel free to do so after consulting the policies
described in [MAINTAINERS.md](./MAINTAINERS.md),
as well as [SECURITY.md](./SECURITY.md), and be sure
to have read the [Code of Conduct](./CODE_OF_CONDUCT.md).

For larger-sized changes, it is recommended that you open
an issue first to discuss the proposed change.

Thank you for contributing to this project.
